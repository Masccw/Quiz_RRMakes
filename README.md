Quiz Temático – RRMakes

Descrição do Projeto

Este projeto consiste em um quiz temático sobre a empresa RRMakes, desenvolvido como trabalho da 2ª Avaliação** da disciplina de Desenvolvimento de Aplicações para Internet.

O quiz foi implementado utilizando apenas HTML5, CSS3 e JavaScript, seguindo os requisitos definidos pelo professor George Alberto, com foco em organização, funcionamento offline, temporizador configurável e apresentação clara do resultado final.

---

 Objetivo

Testar os conhecimentos do usuário sobre a empresa RRMakes por meio de um quiz interativo contendo perguntas de múltipla escolha, com controle de tempo.

---

 Funcionalidades

- Quiz com 10 questões, cada uma com 5 alternativas
- Temporizador configurável pelo usuário antes de iniciar o quiz
- Avanço automático para a próxima questão quando o tempo expira
- Imagem de fundo diferente para cada pergunta
- Exibição do resultado final, contendo:

  - Pontuação total
  - Lista das respostas incorretas
  - Indicação da resposta correta
- Projeto totalmente offline*, sem uso de bibliotecas externas ou CDN

 ▶️ Instruções de Execução

1. Baixe e extraia o arquivo .zip do projeto.
2. Abra a pasta raiz do projeto.
3. Execute o arquivo index.html.
4. O quiz será aberto diretamente no navegador.

 Navegador testado

 Google Chrome
